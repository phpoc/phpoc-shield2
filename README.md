## PHPoC Shield
PHPoC source codes to use [PHPoC (WiFi) Shield for Arduino](http://www.phpoc.com/phpoc_shield_for_arduino.php).

## Available Web Applications
* Setup
* Web Serial Monitor
* Web Serial Plotter
* Web Remote Push
* Web Remote Slide
* Web Remote Pad
